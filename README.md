## Real-Time Camera Detection

The trained YOLO model can be used for real-time barcode detection using your webcam.

### Camera Detection Script

A script is provided to run real-time detection from your webcam:

```bash
python camera_detection.py
```

**Features:**
- Real-time barcode detection from webcam feed
- Displays bounding boxes around detected barcodes
- Shows class labels (1D or 2D) with confidence scores
- FPS counter and barcode count display
- Press `q` to exit

### Photo Capture Detection

To take a photo and then detect barcodes:

```bash
python photo_detection.py
```

**Features:**
- Press `SPACE` to capture a photo
- Automatically runs detection on the captured image
- Displays results with bounding boxes and labels
- Press `q` to exit

### Requirements

- Webcam connected to your MacBook
- Trained YOLO model weights (`best.pt`)
- OpenCV installed (included in `requirements.txt`)

# BarBeR-Project: Barcode Detection and Classification

## Project Overview

This project reproduces the barcode detection task described in the BarBeR paper (ICPR 2024) using the BarBeR dataset. Two approaches were implemented:

1. **Classification** – Binary classification (1D vs 2D barcodes) using a Basic CNN
2. **Detection** – Object detection with bounding boxes using YOLOv8n
3. **Real-Time Camera Detection** – Live barcode detection using webcam

## Dataset

### BarBeR Dataset

- **Total Images:** 8,748
- **Total Annotations:** 9,818 barcodes
- **Classes:** 1D (linear) and 2D (two-dimensional) barcodes
- **Format:** VIA JSON annotations (converted to YOLO format)

### Download Instructions

1. Download the dataset from the official website:
   - **URL:** https://ditto.ing.unimore.it/barber/
   - **Expected Size:** ~5 GB (compressed)

2. Extract the downloaded ZIP file

3. Place the extracted contents in the following structure:
   ```
   data/raw/
   ├── Annotations/
   │   └── VIA/
   │       ├── Artelab.json
   │       ├── Artelab Extended.json
   │       ├── Deal Kaist.json
   │       ├── Muenster.json
   │       ├── SKUU DSP Lab.json
   │       ├── ZVZ-real.json
   │       ├── Open Food Facts.json
   │       ├── InventBar.json
   │       ├── Szentandrasi-Dubska QR.json
   │       ├── Bodnar.json
   │       ├── ParcelBar.json
   │       └── Dubska.json
   └── dataset/
       ├── images/
       │   └── *.jpg (8,748 images)
       └── labels/
           └── *.txt (generated by preprocessing)
   ```

4. After placing the files, run the preprocessing script:
   ```bash
   python scripts/preprocess_data.py --reorganize
   ```

> **Note:** The dataset is large (~5 GB) and is not included in this repository. You must download it manually from the link above.

## Methodology

### Data Preprocessing
- VIA JSON annotations converted to YOLO format
- Polygons converted to normalized bounding boxes
- Train/val/test split: 70%/15%/15%

### Classification Model (BasicCNN)
- 3 convolutional layers + ReLU + MaxPooling
- Adaptive pooling + 2 fully connected layers
- Dropout for regularization
- Trained for 10 epochs on CPU

### Detection Model (YOLOv8n)
- Pre-trained on COCO, fine-tuned on BarBeR
- 130 layers, 3.0M parameters
- Trained for 10 epochs on CPU

## Discussion

The project successfully reproduced the BarBeR detection task with strong performance (97.93% mAP50). The classification model serves as a baseline with 96.34% accuracy. 1D barcodes were detected more accurately than 2D barcodes, likely due to class imbalance in the dataset.

### Limitations
- Class imbalance (82% 1D, 18% 2D)
- Only 10 epochs trained
- CPU-only training limited batch size and speed
- Corrupted images required special handling

### Future Work
- Extended training for more epochs
- Transfer learning with MobileNetV2 or EfficientNet-B0
- GPU acceleration
- Multi-class detection (specific barcode types)
